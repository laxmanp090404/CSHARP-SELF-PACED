# Task-2 Simple OOP App

## Overview

The program:

* Defines a `Person` class
* Uses properties (`Name`, `Age`)
* Implements a method `Introduce()`
* Creates multiple objects and calls their methods

---

## How It Works

1. A `Person` class is created with:

   * `Name` (string)
   * `Age` (int)

2. A method `Introduce()` prints:

```text
Hi, my name is {Name} and I am {Age} years old.
```

3. In `Main()`:

   * Multiple `Person` objects are created
   * Each object calls `Introduce()`

---
