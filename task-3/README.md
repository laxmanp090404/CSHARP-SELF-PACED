# Task-3 List Manager App

## Overview

This is a C# console application that manages a list of items (e.g., names or tasks).

The program:

* Uses `List<string>` to store items
* Allows users to **add**, **remove**, and **display** items
* Uses string manipulation methods to process input

---

## Concepts Used

* Collections (`List<string>`)
* Loops (`while`)
* Conditional statements (`switch`)
* String methods (`Trim()`, `ToUpper()`)

---

## How It Works

1. Displays a menu with options:

   * Add Item
   * Remove Item
   * Display Items
   * Exit

2. User selects an option

3. Based on input:

   * Adds item (converted to uppercase)
   * Removes item (case-insensitive handling)
   * Displays all items

4. Program runs continuously until user exits
